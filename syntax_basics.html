Basic Syntax in CPlusPlus

Let's start with a example

Example:

#include < iostream > using namespace std; int main() { cout<<"Hello World"; return 0; }

Explanation

The above code is a basic template for a CPlusPlus program where

  • #include < include > - is a header file library that lets you do input and output operations
  • using namespace std means that we can use names for objects and variables from the standard library.
  • int main() - is a function which executes code inside flower brackets i.e., "{ }".
  • cout (pronounced "see-out") is an object used together with the insertion operator (<<) to output/print text. In our example it will output "Hello World!".
  • semi-colon (" ; ") in every line indicates end of a line of code.
  • return 0 ends the main function.

Is using namespace std really necessary?

The using namespace std line can be omitted and replaced with the std keyword, followed by the :: operator for some objects.


Comments in CPlusPlus

Comments can be used to explain C++ code, and to make it more readable.

They will not be executed.They are of two types,

  • Single-Lined Comment
  • Multi-Lined Comment

Single-Lined Comment

Single-line comments start with two forward slashes (//).

Any text between // and the end of the line is ignored by the compiler (will not be executed).

Example:

//Are you new to CPlusPlus. cout<<"Hey There!";

Multi-Lined Comment

Multi-line comments start with /* and ends with */.

Any text between /* and */ will be ignored by the compiler

Example:

/* Are you new to CPlusPlus!? Let's make the learning easy!! */ cout<<"Hey There!";
